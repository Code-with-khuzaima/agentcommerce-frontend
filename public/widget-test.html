Widget Test Page

Use this page to test the live AgentComerce chat widget without installing it on a client storefront first.

Ready. Load the widget for a store and test the live chat flow.
What This Tests
  • Live widget config fetch by store_id
  • Welcome text, color, and agent name rendering
  • Chat routing to the saved webhook URL
  • Real message usage tracking after successful replies
How Store URL Usually Works

A client store URL is the website where the snippet is installed, for example https://brand-store.com or a Shopify storefront. For testing, you do not need that. This page acts as a clean storefront substitute.

After This Page Works
  • Give the client their install snippet
  • Client pastes it into Shopify or WooCommerce
  • Save webhook URL in admin
  • Mark the store live after one real message test
